The Test_Sample_nxs_diffraction Instrument

Test TOF-diffraction instrument for the Sample_nxs.comp component

Identification

  • Site: HZB
  • Author: Mirko Boin (HZB)
  • Origin: HZB
  • Date: 2012
  • Version: $Revision$

Description


This instrument relies on the 'nxs' library that is provided with McStas as of release 2.0.
Please see MCSTAS/lib/share/nxslib/README.TXT and in the links section below
for further instructions and more details.

Input parameters

Parameters in boldface are required; the others are optional.
Name Unit Description Default
lambda1 AA Lower bound of wavelength band 1.5
lambda2 AA Upper bound of wavelength band 1.51
sample_detector_distance m Distance between sample and detector 0.1
moderator_pinhole_distance m Distance between moderator and pinhole 30.0
pinhole_sample_distance m Distance between pinhole and sample 0.05
sample_x m Sample translation along x 0.0
sample_file str Crystallographic structure from 'nsx' file "Cu.nxs"
moderator_face str Chosen ISIS moderator "water"
